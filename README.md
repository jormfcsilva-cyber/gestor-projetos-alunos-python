# Gestor de Projetos de Alunos em Python

Aplicação desenvolvida em Python para registo e acompanhamento de projetos de alunos (PAP ou outros trabalhos).

## Funcionalidades

- adicionar projetos
- listar projetos
- pesquisar projetos
- alterar estado
- guardar dados em ficheiro JSON
- estatísticas simples

## Estrutura do projeto

- versao1 – versão inicial da aplicação
- versao2 – melhoria com validação e armazenamento em ficheiro
- versao3 – funcionalidades adicionais e estatísticas
- versão4 - versão com base de dados SQLite (persistência robusta)

## Tecnologias

- Python
- JSON
- SQLite
- programação em consola
